# Kairo Marketplace Analytics Platform

An end-to-end Business Intelligence Engineering project simulating a mid-sized global e-commerce marketplace.

## Project Vision

Build a production-realistic analytics platform for a fictional e-commerce marketplace that mirrors the day-to-day work of a Business Intelligence Engineer — grounded in business questions, powered by trustworthy data, and delivered through actionable analytics.

## Tech Stack

**Local (Phase 1):**
- Python 3.11 for data generation
- Polars & NumPy for high-performance data manipulation
- DuckDB as the local warehouse
- dbt Core for transformations
- Streamlit for dashboards

**Cloud (Phase 2):**
- Amazon S3, Athena, Redshift Serverless
- AWS Glue Data Catalog
- AWS Step Functions & EventBridge
- Amazon QuickSight
